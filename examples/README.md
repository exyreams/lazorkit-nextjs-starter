# Lazorkit Examples

This directory contains working examples demonstrating different aspects of the Lazorkit SDK.

## Available Examples

### [Basic Setup](./basic-setup/)

- Minimal Next.js setup with Lazorkit
- Provider configuration
- Basic wallet connection

### [Authentication](./authentication/)

- Complete passkey authentication flow
- Connect/disconnect functionality
- Session management

### [Transactions](./transactions/)

- SOL transfers with gasless transactions
- USDC transfers with fee payment in USDC
- Message signing for verification

### [Components](./components/)

- Reusable React components
- UI patterns and best practices
- Complete component implementations

## Running Examples

Each example includes its own setup instructions and required polyfills/configuration for the Next.js environment.

Navigate to any example directory and follow the README instructions.
